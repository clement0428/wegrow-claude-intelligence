---
name: Delete buttons on all item lists
description: Every list of items in Orbit UI must have a delete/remove button on each row
type: feedback
---

Every list or table of items in the Orbit frontend MUST include a per-row delete button. No exceptions.

**Why:** User was very frustrated when newly-added pesticide items appeared in the plan table with no way to remove them. This is a fundamental UX expectation.

**How to apply:** Whenever building or modifying any list/table UI (plan rows, task lists, product lists, group items, zone lists, etc.), always include a × or trash icon delete button on each row that immediately removes the item and persists the change.
