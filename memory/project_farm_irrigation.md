# 梓官農場灌溉系統正確架構

## 方案A（田區淹灌 + 果樹精準滴灌）

### 田區路（地下水1）
地下水1 → 地下水馬達1（既有）→ 儲水桶1（既有）→ 灌溉馬達1.5HP → 主幹管1" → 3"電磁閥×7 → 田區7分區畦溝淹灌

### 果樹滴灌路（地下水2+3並聯）
地下水2+3並聯 → 地下水馬達2、3（既有）→ 儲水桶並聯 → RO過濾系統 → 30噸儲水桶 + 5噸液肥桶 → 滴灌馬達1HP×2（主備切換）→ 主幹管1" PE ≈185m → 1"電磁閥×16 → 16mm PE滴灌支管 ≈408m → 四爪滴箭×1,020個（5個/棵×204棵）→ 果樹根部

### 方案A 控制點
1"閥16 + 3"閥7 + 灌溉馬達2 = **25點**

## 方案B（田區淹灌 + 果樹淹灌 雙路）

### 田區路（地下水1）— 與方案A田區路完全相同
地下水1 → 馬達1（既有）→ 桶1（既有）→ 灌溉馬達1.5HP → 3"閥×7 → 田區7分區

### 果樹A路（地下水2）
地下水2 → 10噸桶A（既有）→ **過濾片** → 馬達A 1.5HP → 3"主路閥×1 → 主幹管1" → 1"閥×13 → 13排PVC埋管（寬30-40cm,深20cm）→ 漫流入溝

### 果樹B路（地下水3）
地下水3 → 10噸桶B（既有）→ 馬達B 1.5HP → 3"主路閥×1 → 主幹管1" → 1"閥×12 → 12排PVC埋管 → 漫流入溝

### 方案B 控制點
1"閥25 + 3"閥9（果樹2+田區7）+ 灌溉馬達3（田區1+AB各1）= **37點**

---

## 施工說明（方案B）
- 草苔墊層：溝寬30-40cm，深20cm
- 扒草鏟+網側：灌水慢滲入根部，不要一次灌大砂土
- 方案B淹灌不需RO過濾，省掉整段RO成本
- A路必設過濾片防堵，B路可依需求加裝
- 備援：任一路馬達故障 → 另外兩路可繼續供水

**Why:** 用戶明確說方案B也包含田區，與方案A田區配置一致，之前多次畫錯（漏畫田區路，或田區閥數算錯）
**How to apply:** 畫方案B時永遠3欄：田區路|果樹A路|果樹B路；控制點37點
