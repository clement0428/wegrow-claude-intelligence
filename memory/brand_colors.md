WeGrow 官方識別色（來源：威果wegrow-VIS基礎系統-PDF.pdf）

**主色 Primary — WeGrow Yellow**
- HEX: `#E2CF35`
- RGB: R226, G207, B53
- Pantone: 604 C
- 用途：Sidebar 背景、主要按鈕、強調色、border accent

**輔色 Secondary — Cool Gray**
- HEX: `#808080`
- RGB: R128, G128, B128
- Pantone: Cool Gray 7 C
- 用途：次要文字、輔助元素、inactive 狀態

**配色規則：**
- 底色：白 `#FFFFFF` / 極淺黃灰 `#FAFAF5`
- 主要文字：深炭灰 `#2d2d2d`
- 次要文字：`#808080`
- Hover 加深黃：`#c9b82e`
- 淡黃背景：`#fdf9d0`
- 禁止使用深綠（#1b5e20, #2e7d32）作為品牌主色

**Why:** 用戶明確指出不喜歡舊的深綠配色，要求所有 UI 改用官方 VIS 識別色。
**How to apply:** 所有 Claude 產生的 UI、HTML、dashboard、前端，以及 **Excel 表格、Python openpyxl 輸出**，一律以 #E2CF35 為主色，#808080 為輔色，白底深炭字。Excel 中深色header用 #E2CF35（黑字），次要 section bar 用 #808080，淡色背景用 #FAFAF5 或 #FDF9D0。禁止使用深綠（#1b5e20, #2e7d32）。用戶要求不需提醒，直接套用。
