不要為 wegrow-sales 開發 Roadmap 裡的大部分功能。

**停止開發的項目（全部延後）：**
- LINE/Gmail 一鍵發送
- AI accept/reject 稽核
- Pipeline Kanban 展開
- 報價單 PDF 匯出
- CSV 匯出/匯入
- 投資人獨立 Tab
- Dashboard 時間序列圖
- Notion 同步
- Gmail → CRM 自動更新
- 每日 08:00 LINE Briefing
- RWD 手機版
- 美化/體驗優化

**Why:** 這些功能「看起來完整」但不會讓業務更快成交。資料量小、流程未驗證時，自動化和 AI 功能變成裝飾。

**唯一值得開發的最小項目：**
Pipeline + next action + follow-up reminder

**CRM 只需要做到：**
1. 看潛在客戶名單
2. 記錄下一步 action
3. 標記 stage / priority
4. 追蹤誰需要 follow-up
5. 手動匯出或複製給團隊

**How to apply:** 每次有人提議為 wegrow-sales 加新功能時，先問「這會讓今天多追一個客戶、少漏一個 follow-up、提高成交率嗎？」答案是否就延後。
